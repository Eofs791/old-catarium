## Java Monster Irish Crème
喝之前坐公交了，晕晕的。
下午三点十分开始喝，二十分钟左右干完。
喝的时候感觉左手有点抖（可能是太冷了），喝完还是晕。
棕色，口味就是普通牛奶咖啡，有点甜味，无气泡。
当天晚上睡眠还行，11点多睡着3点半醒了一次。

## Java Monster Mean Bean
下午两点开始喝，20分钟左右干完。
棕色，口味也是普通牛奶咖啡，比Irish Crème更甜更丝滑一点。
当天晚上0点才睡着（平常10点躺11点左右就睡了），也可能是一直刷手机导致，不一定是咖啡功效。

## Java Monster Loca Moca
十二点二十分开始喝，两点下课喝完。   
灰卡其色，巧克力味更浓郁一些。   
上课时感觉略微精神。
